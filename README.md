# RainVisz
An audio visualizer skin for [Rainmeter](https://www.rainmeter.net).

## Configuration
For configuring the skin modify the variables of the [``variables.ini``](%40Resources/variables.ini) file in the [``@Resources``](%40Resources) folder.
